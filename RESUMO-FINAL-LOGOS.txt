🌐 SE ESTIVER USANDO SEU DOMÍNIO REGISTRO.BR:

1. Copie HTML + imagens para seu servidor

   └─ Se usar Vercel:

      • ns4.vercel-dns.com

      • dns2.p04.nsone.net
      • dns3.p04.nsone.net
      • dns4.p04.nsone.net

5. Aguarde 2-48h para propagação DNS
   └─ Verificar em: https://dnschecker.org

📞 PROBLEMAS FREQUENTES:

P: As logos não aparecem!
R: Verifique se as imagens estão no mesmo diretório que o HTML.
   Se deployer no Vercel/Netlify, precisa fazer upload dos PNGs também.

P: Logos parecem pixeladas/distorcidas
R: Zoom do navegador? (Ctrl+0 para resetar)
   Ou é normal de JPG grande - melhora após deploy.

P: Está lento em meu computador
R: PNGs de 2MB são grandes. Após deploy em CDN (Vercel), fica rápido.
   Localmente pode demorar mais (especialmente primeira vez).

P: Deploy deu erro
R: Certifique-se:
   1. HTML + 9 PNGs fizeram upload
   2. Estão no mesmo diretório
   3. Nomes de arquivo estão corretos (case-sensitive)
   4. Sem caracteres estranhos nos nomes
