📞 SE ALGO DER ERRADO:

Problema: Logos não aparecem
Solução: 
1. Verifique se PNGs estão no mesmo diretório que HTML
2. Verifique se nomes de arquivo estão exatos (case-sensitive)
3. Verificar console (F12 > Console) para erros de 404
4. Se não houver error, recarregue a página (Ctrl+F5)

Problema: Logos parecem distorcidas
Solução:
1. Isso é normal com imagens JPG grandes
2. Verificar zoom do navegador (Ctrl+0)
3. Tentar em navegador diferente

Problema: Site lento com as imagens
Solução:
1. As imagens são grandes (~2MB cada)
2. Vercel/Netlify comprime automaticamente
3. Performance melhora após deploy em CDN
4. Localmente pode ser um pouco mais lento

💡 DICAS FINAIS:

1. Os nomes de arquivo são longos
   └─ Você pode renomear para: hackstartup.png, hackclub.png, etc
   └─ Mas precisa atualizar HTML também

2. Se quiser otimizar para web
   └─ Use https://tinypng.com para comprimir PNGs
   └─ Pode reduzir de 2MB para 500KB sem perder qualidade

3. Considere usar imagens WebP
   └─ Ainda menor (200-300KB)
   └─ Compatível com navegadores modernos
   └─ Fallback para PNG em navegadores antigos

4. Para produção profissional
   └─ Hospede imagens em CDN (Cloudinary, Imgix)
   └─ Melhora performance drasticamente
   └─ Lazy loading funciona melhor
